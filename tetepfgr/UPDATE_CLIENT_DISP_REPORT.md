**После обновления:**
- **6 шагов** с полным описанием функционала
- Добавлены ВСЕ функции из client-disp-dev:

#### Новые шаги:

1. **Выбор смены и даты**
   - Фильтр смен с выбором даты
   - Выбор номера смены (1, 2, 3)
   - Автоматическая загрузка техники
   - Карточки для каждой единицы техники

2. **Фильтрация техники**
   - Добавление/удаление техники из фильтра
   - Работа с подмножеством машин
   - Подтверждение при удалении с несохраненными данными

3. **Создание наряд-заданий**
   - Заполнение параметров для каждой техники:
     - Маршрут (место погрузки → разгрузки)
     - Тип груза
     - Объем кузова
     - Количество рейсов
     - Плановый объем (авторасчет)
     - Плановый вес (авторасчет)
   - Автосохранение при вводе

4. **Копирование из предыдущей смены** ⭐ КРИТИЧЕСКАЯ ФУНКЦИЯ
   - Кнопка на панели инструментов
   - Копирование всех данных кроме отмененных
   - Подтверждение при наличии сохраненных заданий
   - Сообщение об успехе с количеством

5. **Массовые операции**
   - Очистка данных (только "К заполнению")
   - Фильтрация по статусу ("В работе" / "Все")
   - Сводка по общему плановому объему

6. **Валидация и отправка**
   - Проверка обязательных полей
   - Проверка корректности маршрутов
   - Предупреждения с типами ошибок
   - Подтверждение отправки с предупреждениями
   - Сохранение в БД
   - Видимость на бортовых терминалах
