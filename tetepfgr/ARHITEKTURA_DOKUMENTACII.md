# Архитектура документации АСУ ПГР

## Общая структура

```
ДОКУМЕНТАЦИЯ АСУ ПГР
│
├─ Быстрый старт
├─ Администрирование
├─ Обзор системы
├─ Пошаговые инструкции
├─ Справочники
└─ О системе
```

---

## 1. Быстрый старт

**Файл:** `data.json` → `cards.quickstart`

**Назначение:** Первое знакомство с системой

**Целевая аудитория:** Пользователи (диспетчеры, операторы)

**Содержание:**

```
Шаг 1. Вход в систему
├─ Авторизация
└─ Главный экран

Шаг 2. Знакомство с главным меню
├─ Обзор интерфейса
└─ Основные разделы меню

Шаг 3. Проверка готовности системы
├─ Проверка справочников
└─ Что делать если справочники пустые

Шаг 4. Создание наряд-заданий
├─ Выбор смены
├─ Создание маршрутного задания
└─ Копирование из предыдущей смены

Шаг 5. Мониторинг работы
├─ Просмотр рейсов
├─ Работа с картой рудника
└─ Анализ карты рабочего времени (КРВ)

Шаг 6. Работа с бортовым терминалом
├─ Главный экран оператора
├─ Управление статусами техники
├─ Регистрация простоев
├─ Просмотр статистики смены
└─ Работа с наряд-заданиями
```

**Особенности:**
- ✅ Предполагается что система УЖЕ настроена администратором
- ✅ Справочники УЖЕ заполнены (оборудование, места, горизонты, виды груза, статусы)
- ✅ Пользователь РАБОТАЕТ с готовой системой (не настраивает её)
- ✅ Если справочники пустые → отсылка к разделу "Администрирование"
- ✅ Фокус на ежедневной работе диспетчера: планирование смены и мониторинг выполнения

---

## 2. Администрирование

**Файл:** `data.json` → `cards.admin_instructions`

**Назначение:** Инструкции по настройке системы с нуля

**Целевая аудитория:** Администраторы

**Содержание:**

```
Шаг 1. Вход в систему
Шаг 2. Знакомство с главным меню
Шаг 3. Настройка видов груза
Шаг 4. Настройка статусов
Шаг 5. Настройка ролей
Шаг 6. Настройка горизонтов
Шаг 7. Настройка участков
Шаг 8. Настройка мест работ
Шаг 9. Настройка меток позиционирования
Шаг 10. Настройка оборудования
Шаг 11. Настройка персонала
```

**Особенности:**
- ✅ Пошаговая настройка системы с нуля
- ✅ Порядок шагов учитывает зависимости между справочниками
- ✅ Каждый шаг содержит подробные инструкции с описанием полей
- ✅ Скриншоты для каждого действия
- ✅ Предупреждения о важных моментах

**Целевая аудитория:** Все

**Содержание:**

```
├─ Единицы оборудования (vehicles)
├─ Статусы (statuses)
├─ Виды груза (cargo_types)
├─ Места работ (work_places)
├─ Горизонты (horizons)
├─ Метки позиционирования (positioning_tags)
├─ Наряд-задания (work_orders)
├─ Рейсы (trips)
└─ Участки (sections)
```

## Принципы организации

### 1. Разделение по аудитории
- ✅ Пользователи → Быстрый старт + Пошаговые инструкции
- ✅ Администраторы → Администрирование
- ✅ Все → Справочники + О системе

### 2. От простого к сложному
- ✅ Быстрый старт → краткий обзор
- ✅ Инструкции → детальные шаги
- ✅ Справочники → полное описание параметров

### 3. Контекстные ссылки
- ✅ Из Быстрого старта → на Инструкции
- ✅ Из Инструкций → на Справочники
- ✅ Из Справочников → на Администрирование
