# Архитектура документации АСУ ПГР

## Общая структура

```
ДОКУМЕНТАЦИЯ АСУ ПГР
│
├─ Быстрый старт
├─ Администрирование
├─ Обзор системы
├─ Пошаговые инструкции
├─ Справочники
└─ О системе
```

---

## 1. Быстрый старт

**Файл:** `data.json` → `cards.quickstart`

**Назначение:** Первое знакомство с системой

**Целевая аудитория:** Пользователи (диспетчеры, операторы)

**Содержание:**

```
Шаг 1. Вход в систему
├─ Авторизация
└─ Главный экран

Шаг 2. Обзор интерфейса
├─ Навигационное меню
├─ Верхняя панель
└─ Рабочая область

Шаг 3. Проверка справочников
├─ Как открыть справочники
├─ Что должно быть заполнено
├─ Проверка готовности системы
└─ Что делать если справочники пустые

Шаг 4. Создание наряд-задания
├─ Выбор смены
├─ Создание маршрутного задания
└─ Активация задания

Шаг 5. Мониторинг работы
├─ Просмотр рейсов
├─ Карта рудника
└─ Карта рабочего времени (КРВ)
```

**Особенности:**
- ✅ Предполагается что система УЖЕ настроена администратором
- ✅ Справочники УЖЕ заполнены
- ✅ Пользователь РАБОТАЕТ с готовой системой
- ✅ Если справочники пустые → отсылка к разделу "Администрирование"

---

## 2. Администрирование

**Файл:** `data.json` → `cards.admin_instructions`

**Назначение:** Инструкции по настройке системы с нуля

**Целевая аудитория:** Администраторы

**Содержание:**

```
├─ Первичная настройка системы
├─ Настройка оборудования
├─ Настройка справочников
├─ Управление пользователями
└─ Настройка системы
```

---

## 3. Обзор системы

**Файл:** `data.json` → `cards.descriptive`

**Назначение:** Общее описание разделов и возможностей системы

**Целевая аудитория:** Все

**Содержание:**

```
├─ Справочники
├─ Оперативная работа
├─ Отчеты
├─ Диагностика
└─ Настройки
```

---

## 4. Пошаговые инструкции

**Файл:** `data.json` → `cards.instructions`

**Назначение:** Детальные инструкции по работе с системой

**Целевая аудитория:** Пользователи (диспетчеры, операторы)

**Содержание:**

```
├─ Работа со справочниками
├─ Наряд-задания
├─ Мониторинг
├─ Отчеты
└─ Бортовой терминал
```

---

## 5. Справочники

**Файл:** `directory_data.json` → `directories`

**Назначение:** Подробное описание каждого параметра справочников

**Целевая аудитория:** Все

**Содержание:**

```
├─ Единицы оборудования (vehicles)
├─ Статусы (statuses)
├─ Виды груза (cargo_types)
├─ Места работ (work_places)
├─ Горизонты (horizons)
├─ Метки позиционирования (positioning_tags)
├─ Наряд-задания (work_orders)
├─ Рейсы (trips)
└─ Участки (sections)
```

---

## 6. О системе

**Файл:** `data.json` → `cards.about`

**Назначение:** Общая информация о системе

**Целевая аудитория:** Все

**Содержание:**

```
├─ Назначение системы
├─ Архитектура
├─ Роли и права доступа
└─ Техническая поддержка
```

---

## Принципы организации

### 1. Разделение по аудитории
- ✅ Пользователи → Быстрый старт + Пошаговые инструкции
- ✅ Администраторы → Администрирование
- ✅ Все → Справочники + О системе

### 2. От простого к сложному
- ✅ Быстрый старт → краткий обзор
- ✅ Инструкции → детальные шаги
- ✅ Справочники → полное описание параметров

### 3. Контекстные ссылки
- ✅ Из Быстрого старта → на Инструкции
- ✅ Из Инструкций → на Справочники
- ✅ Из Справочников → на Администрирование
