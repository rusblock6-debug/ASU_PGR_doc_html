## Структура бортового терминала (из кода)

### Маршруты (routes):
- / - MainScreen (главный экран)
- /work-orders - список наряд-заданий
- /work-orders/:taskId - детали задания
- /vehicle-status - управление статусами
- /downtime-select - выбор причины простоя
- /active-downtime - активный простой
- /stats - статистика смены

### Ключевые виджеты:
- TripGauge - счетчик рейсов
- StatusBar - панель статуса
- ActiveRoutePanel - панель активного маршрута
- BottomNav - нижняя навигация
- RouteTaskList - список заданий
- RouteTaskDetailScreen - детали задания
- ConfirmButton - кнопка подтверждения
- KioskBackButton - кнопка назад

### API вызовы:
- useCurrentShiftTasks - получить задания смены
- useGetActiveTaskQuery - активное задание
- useSetVehicleStateTransitionMutation - смена статуса
- useGetCurrentShiftStatsQuery - статистика смены
- useRoutesStreamSse - SSE поток маршрутов
