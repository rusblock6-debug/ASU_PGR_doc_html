# Документация - Цифровой двойник карьера

Интерактивная документация для системы имитационного моделирования карьера.

## Структура проекта

```
├── admin.html              # Админ-панель с редактированием
├── data.json               # Данные карточек
├── directory_data.json     # Справочники
├── server.js               # Node.js сервер
├── screenshots/            # Изображения (создается автоматически)
└── README.md               # Документация
```

## Версии

**admin.html** - Полная версия с редактированием. Требует локальный сервер Node.js.

## Запуск

Требования: Node.js (любая версия)

```bash
node server.js
```

Доступ:
- Админка: http://localhost:3000/admin.html

## Функции

**Карточки документации:**
- Создание и редактирование карточек (Обзор, Инструкции, О системе)
- Загрузка и управление изображениями
- Автоматическая организация скриншотов по папкам
- Удаление карточек с полной очисткой файлов

**Справочники:**
- Создание справочников параметров
- Редактирование и удаление справочников
- Табличное отображение данных

**Интерфейс:**
- Глобальный поиск по всему контенту
- Экспорт карточек в PDF
- Адаптивный дизайн для мобильных устройств
- Навигация с автоматическим оглавлением

## Развертывание

**Docker:**
```dockerfile
FROM node:alpine
WORKDIR /app
COPY . .
EXPOSE 3000
CMD ["node", "server.js"]
```

**Прямо на сервере:**
Скопировать все файлы и запустить `node server.js`

**GitHub Pages:**
Переименовать нужную версию в index.html и загрузить в репозиторий

## API

- GET /api/data - получение данных
- POST /api/data - сохранение данных
- POST /api/save-image - загрузка изображения
- POST /api/delete-image - удаление изображения
- POST /api/delete-folder - удаление папки карточки
- POST /api/save-directory - сохранение справочников

## Структура данных

**data.json** - основные карточки документации
**directory_data.json** - справочники параметров
**screenshots/** - изображения, организованные по типам:
- overview/ - изображения карточек обзора
- instructions/ - изображения инструкций  
- about/ - изображения карточек "О системе"
- fast/, 1st_slide/, 2nd_slide/ - существующие изображения

## Особенности

- Нет внешних зависимостей (только стандартный Node.js)
- Автоматическая транслитерация имен файлов
- Кроссплатформенность (Windows/Linux/macOS)
- Минимальные требования к ресурсам
- Полная очистка при удалении карточек
