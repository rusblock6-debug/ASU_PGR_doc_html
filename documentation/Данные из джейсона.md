
## Быстрый старт

### Авторизация

1. Перейдите по адресу системы
2. Введите логин и пароль
3. Нажмите «Войти»

### Главное меню

#### Обзор интерфейса

1. Логотип — переход на главную
2. Кнопка «Домой» — рабочий стол
3. Кнопка «Виджеты» — переход к закрепленным страницам
4. Кнопка «Меню» — открытие навигации
5. Дата, время, часовой пояс
6. Статус подключения
7. Центральная область с карточками
8. Раздел «Оперативная работа»
9. Раздел «Отчеты»
10. Раздел «Справочники»
11. Раздел «Диагностика»
12. Раздел «Настройки»
13. Меню пользователя

### Проверка готовности

#### Проверка справочников

1. Откройте «Справочники»
2. Проверьте заполнение разделов:
   - Виды груза
   - Статусы
   - Горизонты
   - Места
   - Оборудование
3. Переходите к созданию заданий

#### Если справочники пустые

1. Обратитесь к администратору
2. Администратор заполнит справочники
3. Обновите страницу

Без справочников невозможно создать задания

### Создание заданий

#### Выбор смены

1. Откройте «Оперативная работа» → «Наряд-задание»
2. Выберите смену в фильтре
3. Дождитесь загрузки списка техники

#### Создание маршрута

1. Найдите карточку техники
2. Нажмите «+»
3. Заполните маршрут:
   - Место погрузки
   - Место разгрузки
   - Объем/масса (при необходимости)
4. Нажмите «Отправить»

#### Копирование из предыдущей смены

Используйте для повторения заданий из предыдущей смены.

1. Нажмите «Скопировать из предыдущей смены»
2. Проверьте скопированные задания
3. Нажмите «Отправить»

### Мониторинг

#### Управление рейсами

1. Откройте «Оперативная работа» → «Управление рейсами»
2. Выберите смену
3. Добавьте рейс:
   - Нажмите «Добавить»
   - Заполните форму
   - Нажмите «Сохранить»
4. Редактируйте рейс:
   - Дважды кликните по рейсу
   - Измените данные
   - Нажмите «Сохранить»

#### Карта 2D

Используйте для мониторинга техники и редактирования карты рудника.

1. Откройте «Оперативная работа» → «Карта (2D)»
2. Выберите горизонт
3. Выберите режим:
   - Просмотр — мониторинг техники
   - Редактирование — изменение карты
   - История — воспроизведение перемещений
4. Настройте отображение в панелях «Объекты» и «Слои карты»

#### Карта рабочего времени

1. Откройте «Оперативная работа» → «КРВ»
2. Выберите технику и дату
3. Изучите временную шкалу (цветные блоки = статусы)
4. Редактируйте (правый клик мышью по цветному блоку):
   - Назначить новый статус
   - Изменить статус
   - Разделить статус
   - Удалить

#### 3D визуализация

Используйте для 3D мониторинга техники и дорог рудника.

1. Откройте «Оперативная работа» → «3D визуализация Шахты»
2. Просмотр техники:
   - Нажмите кнопку с иконкой транспорта справа
   - Выберите технику из списка
   - Нажмите «Центрировать на карте»
3. Управление камерой:
   - ЛКМ + перемещение — поворот
   - Колесо мыши — приближение/отдаление
   - ПКМ + перемещение — перемещение
4. Кнопки вида:
   - Сверху / Спереди / Сбоку / Изометрия
   - Ближе / Дальше — приближение/отдаление камеры

## Быстрый старт (Борт)

### Вход в систему

#### Авторизация

После включения терминала откроется экран входа.

1. Введите учетные данные:
   - Логин — введите ваше имя пользователя
   - Пароль — введите пароль (символы скрыты)
2. Управление с помощью манипулятора:
   - Стрелки ↑ ↓ — переключение между полями ввода
   - Кнопка подтверждения — выбор активного поля
   - После выбора поля используйте кнопки для ввода символов
3. Нажмите кнопку «ВОЙТИ»

Или нажмите кнопку подтверждения на манипуляторе.

При ошибке появится сообщение «Неверный логин или пароль»

### Главное меню

#### Дополнительные функции

Используйте главное меню для доступа к дополнительным функциям терминала.

1. Откройте главное меню:
   - Коснитесь иконки [☰] в нижнем меню (крайняя правая кнопка)
2. Сетка 2×2 с 4 кнопками:
   - ТЕКУЩЕЕ ЗАДАНИЕ — возврат на главный экран
   - ЖУРНАЛ СОБЫТИЙ — история действий оператора
   - НАСТРОЙКИ — параметры терминала
   - СТАТИСТИКА — просмотр статистики смены

### Наряд-задания

#### Список наряд-заданий

Просмотрите все назначенные маршруты на текущую смену.

1. Перейдите к списку наряд-заданий:
   - Коснитесь иконки [✓] в нижнем меню (третья кнопка слева)
2. Таблица маршрутов содержит:
   № | Начало | Конец | Тип груза | Рейсы | Вес | Начало | Оконч. | Статус
3. Просмотр деталей задания:
   - Коснитесь нужного маршрута в таблице
   - Откроется экран с кнопками управления заданием

#### Управление заданием

##### Кнопки действий

Используйте кнопки управления на экране деталей задания.

1. Кнопки управления:
   - [ОТМЕНИТЬ] — отменить задание (доступна для назначенных заданий)
   - [ПАУЗА] — временно приостановить (доступна для активных заданий)
   - [ПРИСТУПИТЬ / ЗАВЕРШИТЬ] — основная кнопка:
     - ПРИСТУПИТЬ — начать выполнение (для неактивных заданий)
     - ЗАВЕРШИТЬ — досрочно закончить (для активных заданий)
2. Управление кнопками:
   - Коснитесь нужной кнопки на сенсорном экране
   - Неактивные кнопки отображаются серым цветом

Кнопка ПРИСТУПИТЬ возвращает на главный экран

### Главный экран

#### Обзор интерфейса

После входа в систему откроется главный экран бортового терминала.

1. Элементы главного экрана:
   - Счетчик рейсов (левый верхний угол) — прогресс: ФАКТ / ПЛАН
   - Статус техники (левый нижний угол) — текущий статус и таймер
   - Активный маршрут (правый верхний угол) — ОТКУДА → КУДА
   - Нижнее меню (внизу) — 4 кнопки навигации
2. Нижнее меню содержит:
   - ⚙️ Настройки — параметры терминала
   - 🕐 История — журнал событий
   - ✓ Наряд-задания — список маршрутов смены
   - ☰ Меню — дополнительные функции
3. Управление с помощью манипулятора:
   - Коснитесь элемента на сенсорном экране для перехода

### Переключение статусов

#### Как переключать статусы

Переключайте статусы техники по мере выполнения рейса.

1. Перейдите к экрану статусов:
   - Коснитесь панели статуса слева внизу
   - Или коснитесь иконки 🕐 в меню
2. Цепочка из 6 статусов:
   - ДВИЖЕНИЕ ПОРОЖНИМ — едете к погрузке
   - СТОЯНКА ПОРОЖНИМ — ждете погрузки
   - ПОГРУЗКА — загружаетесь
   - ДВИЖЕНИЕ ГРУЖЁНЫМ — едете к разгрузке
   - СТОЯНКА ГРУЖЁНЫМ — ждете разгрузки
   - РАЗГРУЗКА — выгружаетесь
   (Выполненные статусы подсвечены зеленой полосой)
3. Переключение статусов:
   - После завершения действия нажмите [→] СТАТУС ВПЕРЕД (вверху слева)
   - Коснитесь кнопки на сенсорном экране для подтверждения
   (Нельзя пропускать статусы. Кнопка неактивна на последнем статусе)

💡 Кнопка [⏸] ПРОСТОИ (вверху справа) — для регистрации простоя

### Регистрация простоев

#### Как зарегистрировать простой

Если техника простаивает, зарегистрируйте причину простоя.

1. Перейдите к экрану выбора причин:
   - На экране статусов коснитесь [⏸] ПРОСТОИ (вверху справа)
2. Выберите причину простоя:
   Страница 1:
   - РАБОЧИЕ ЗАДЕРЖКИ
   - ЭНЕРГЕТИЧЕСКИЙ ПРОСТОЙ
   - ВСПОМОГАТЕЛЬНЫЕ РАБОТЫ
   - ЭКОЛОГО-КЛИМАТИЧЕСКИЕ
   - РЕГЛАМЕНТИРУЕМЫЕ ПЕРЕРЫВЫ
   - РЕЗЕРВ
   - ТЕХНОЛОГИЧЕСКИЕ ПРОСТОИ
   Страница 2 (кнопка «СЛЕДУЮЩАЯ СТРАНИЦА →»):
   - ПЕРЕВОЗКА ЩЕБНЯ / СНЕГА
   - РАБОТА НА ХОЗ КАРЬЕРАХ / ХВОСТАХ
   - ПЕРЕВОЗКА ВНУТРИСКЛАД
   - ИСПЫТАНИЯ НОВОГО ОБОРУДОВАНИЯ
3. Подтвердите выбор:
   - Коснитесь нужной причины на сенсорном экране
   - Откроется экран с таймером простоя
4. Завершение простоя:
   - Когда простой закончился, нажмите «ЗАВЕРШИТЬ ПРОСТОЙ»
   - Система вернет предыдущий статус техники

Всегда регистрируйте простои

### Статистика смены

#### Просмотр статистики

Просмотрите статистику выполненной работы за текущую смену.

1. Перейдите к экрану статистики:
   - На главном экране коснитесь иконки 📊 (левый верхний угол счетчика)
2. Верхний блок — 6 показателей:
   - ЭФ.ВРЕМЯ, МИН
   - ПРОСТОИ, МИН
   - ПУТЬ, КМ (сейчас: —)
   - РЕЙСЫ, ШТ. — выполнено/план
   - ТОПЛИВО, Л (сейчас: —)
   - ВЕС, Т — факт/план
3. Нижний блок — таблица маршрутов:
   Таблица содержит 9 колонок:
   № | Начало | Конец | Тип груза | Рейсы | Вес | Начало | Оконч. | Статус
   Цвета статусов:
   - 🟢 Зеленый — выполнен
   - 🔵 Синий — в работе
   - ⚪ Серый — ожидает
   - 🔴 Красный — отменен
   - 🟡 Желтый — на паузе

## Обзор

### Справочники

Основной раздел для настройки цифровой модели рудника. Здесь создаются и редактируются все объекты системы.

- Горизонты — уровни рудника с высотными отметками
- Места — точки погрузки и разгрузки
- Участки — производственные участки предприятия
- Метки — метки позиционирования для техники
- Персонал — сотрудники предприятия и их учетные записи
- Роли — роли пользователей с настройкой прав доступа
- Оборудование — модели техники и единицы оборудования
- Статусы — статусы для классификации рабочего времени
- Виды груза — типы материалов с параметрами плотности

### Оперативная работа

Раздел для ежедневного планирования и мониторинга. Основное рабочее место диспетчера.

- Управление техникой — мониторинг состояния и управление парком техники
- Карта — 3D визуализация рудника с техникой в реальном времени
- Карта (2D) — 2D отображение рудника с техникой и местами
- Карта рабочего времени (КРВ) — визуализация статусов техники на временной шкале
- Управление рейсами — просмотр и редактирование текущих и завершенных рейсов
- Наряд-задание — создание сменных заданий для техники

### Отчеты

Раздел для анализа производительности и формирования отчетности.

- Сменная статистика для оборудования
- Эффективность техники
- Лог действий пользователей
- Архив сообщений

### Диагностика

Раздел для мониторинга технического состояния оборудования и системы.

- Диагностика оборудования
- Диагностика системы

### Настройки

Раздел для настройки параметров системы и предприятия.

- Настройки предприятия
- Системные настройки

## Пошаговые инструкции

### Как персонализировать рабочую область?

#### Настройка быстрого доступа к часто используемым страницам

##### Шаг 1

В чем разница между закреплением и избранным?

📌 Закрепление (Рабочая область):
- Страница отображается как виджет на отдельной странице «Рабочая область»
- Виджет показывает полное содержимое страницы (можно работать прямо в виджете)
- Можно перемещать и изменять размер виджетов
- Ограничение: до 4 страниц
- Доступ: через кнопку «Рабочая область» в верхнем меню

⭐ Избранное (Главная страница):
- Страница отображается как ссылка в разделе «Избранное» на главной странице
- Это просто быстрая ссылка для перехода на страницу
- Нет ограничений по количеству
- Доступ: на главной странице системы

Можно использовать оба способа одновременно для одной страницы!

##### Шаг 2

Закрепление страницы

1. Откройте любую страницу системы (например, «Наряд-задание»)
2. В правом верхнем углу найдите кнопку с иконкой булавки 📌
3. Нажмите на кнопку

Или можно закрепить страницу через меню:
- Нажмите кнопку «Меню» в левом верхнем углу
- Найдите нужную страницу в списке разделов
- Нажмите на иконку 📌 рядом с названием страницы

Страница будет закреплена на рабочей области.

Чтобы открепить страницу:
- Вернитесь на страницу
- Нажмите кнопку 📌 повторно
- Или откройте меню и нажмите 📌 рядом с названием страницы
- Страница исчезнет с рабочей области

##### Шаг 3

Добавление в избранное

1. Откройте страницу, которую хотите добавить в избранное
2. В правом верхнем углу нажмите кнопку с иконкой звезды ⭐
3. Страница добавится в список избранных на главной странице

Чтобы удалить из избранного:
- Нажмите кнопку ⭐ повторно
- Страница удалится из списка

##### Шаг 4

Работа с рабочей областью

1. Откройте рабочую область через кнопку в верхнем меню
2. Вы увидите сетку с закрепленными виджетами (до 4 страниц)
3. Каждый виджет — это закрепленная страница

Перемещение виджетов:
- Зажмите заголовок виджета мышью
- Перетащите в нужное место сетки
- Отпустите кнопку мыши

Удаление виджета:
- Нажмите кнопку 📌 в заголовке виджета
- Или откройте меню и снимите закрепление со страницы
- Виджет удалится с рабочей области

##### Шаг 5

Изменение размеров виджетов

1. Наведите курсор на любой угол или сторону виджета
2. Курсор изменится на стрелку изменения размера
3. Зажмите левую кнопку мыши и тяните
4. Отпустите кнопку мыши — размер сохранится автоматически

#### Ключевые показатели:

- Закрепление страниц на рабочей области через кнопку булавки
- Добавление страниц в избранное через кнопку звезды
- Управление виджетами: перемещение, изменение размера, удаление
- Персонализация интерфейса для быстрого доступа к разделам

## О системе

### О руководстве

Руководство пользователя АСУ ПГР — автоматизированной системы управления подземными горными работами. Содержит инструкции по работе с системой для диспетчеров, операторов и администраторов.

#### Структура документа

- Быстрый старт — первые шаги для начала работы с системой
- Администрирование — настройка справочников и управление пользователями
- Обзор системы — описание основных разделов и интерфейса
- Пошаговые инструкции — детальные руководства по работе с функциями системы
- Параметры справочников — описание параметров и полей всех справочников системы

#### Основные возможности

- Управление справочниками: оборудование, персонал, роли, места работ, горизонты, участки, метки, статусы, виды груза
- Создание и управление наряд-заданиями для техники
- Отслеживание перемещения техники в реальном времени
- Визуализация работы на 2D и 3D картах рудника
- Управление рейсами техники
- Анализ производительности через карту рабочего времени (КРВ)
- Управление техникой и контроль выполнения заданий

#### Навигация

Используйте меню слева для перехода между разделами. Инструкции содержат скриншоты интерфейса.

## Администрирование

### Вход в систему

1. Откройте браузер (рекомендуется Google Chrome или Mozilla Firefox)
2. Перейдите на главную страницу экосистемы цифровых решений:
   На экране отображается список доступных систем. Найдите карточку «АСУ ПГР» с описанием «автоматизированная система управления подземными горными работами».
3. Нажмите на карточку «АСУ ПГР»
   Откроется всплывающее окно «Авторизация».

### Настройка видов груза

Виды груза — это типы материалов, которые перевозит техника. Они используются для классификации перевозимых материалов, автоматического расчета веса груза по объему через плотность, контроля соответствия груза между местами погрузки и разгрузки, формирования отчетов по видам перевезенного груза.

1. В разделе «Справочники» выберите пункт «Виды груза»
   Откроется страница со списком видов груза.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания нового вида груза.
3. Заполните поля формы:
   - В поле «Наименование вида груза» введите название материала. Отображается в таблице видов груза, в списках при создании мест погрузки, в наряд-заданиях
   - В поле «Категория вида груза» выберите категорию из списка или создайте новую. Используется для группировки видов груза, автоматически заполняет поле «Является полезным ископаемым»
   - В поле «Плотность, кг/м³» укажите плотность материала в килограммах на кубический метр. Критически важно для автоматического расчета веса груза. Используется для расчета веса в наряд-заданиях, при перемещении техники, для контроля загрузки. Без плотности система не сможет рассчитать вес и заблокирует сохранение наряд-задания
   - В поле «Является полезным ископаемым» выберите «Да» или «Нет». Автоматически заполняется при выборе категории, но можно изменить вручную. Используется для классификации материалов и формирования отчетов
   - В поле «Цвет» выберите цвет для визуализации вида груза. Нажмите на поле выбора цвета, выберите цвет из палитры или введите код в формате HEX. Отображается в таблице видов груза как цветной квадрат
4. Нажмите кнопку «Сохранить»
   Вид груза будет добавлен в таблицу, появится сообщение об успешном создании.
5. Проверьте созданный вид груза в таблице:
   В таблице отображаются колонки: Наименование вида груза, Категория вида груза, Плотность, Является полезным ископаемым, Цвет.
6. Повторите шаги 2-4 для создания всех видов груза:
   Создайте виды для всех материалов, которые перевозятся на предприятии.

ВАЖНО:
- Плотность критически важна для расчета веса груза
- Без правильной плотности система не сможет корректно рассчитать вес
- Плотность должна соответствовать реальным характеристикам материала
- Вид груза места погрузки должен соответствовать виду груза места разгрузки в маршруте
- Виды груза используются для формирования отчетов по перевезенным материалам

### Настройка статусов

Статусы — это состояния техники в процессе работы, которые используются для классификации времени. Они применяются для учета рабочего времени и простоев техники, расчета коэффициента использования основного времени, анализа эффективности работы техники, формирования отчетов по использованию времени.

1. В разделе «Справочники» выберите пункт «Статусы»
   Откроется страница со списком статусов. Система содержит предустановленные системные статусы.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания нового статуса.
3. Заполните поля формы:
   - В поле «Отображаемое название» введите название статуса. Отображается в списке статусов, на карте диспетчера, на карте рабочего времени, в подсказках при наведении на технику
   - В поле «Цвет» выберите цвет для визуализации статуса. Нажмите на поле выбора цвета, выберите цвет из палитры или введите код в формате HEX. Цвет используется для отображения статуса на карте диспетчера, на карте рабочего времени, в списке статусов
   - В поле «Аналитическая категория» выберите категорию для группировки статусов. Используется для группировки статусов в аналитике и отчетах, отображается в таблице статусов
   - Установите галочку «Является рабочим статусом» если время в этом статусе считается рабочим. Определяет учитывается ли время в расчете коэффициента использования. Для системных статусов это поле нельзя изменить
   - В поле «Организационная категория» выберите категорию из списка или создайте новую. Используется для дополнительной группировки статусов по организационным признакам. Можно создать новую категорию прямо на странице статусов
4. Нажмите кнопку «Сохранить»
   Статус будет добавлен в таблицу, появится сообщение об успешном создании.
5. Проверьте созданный статус в таблице:
   В таблице отображаются колонки: Наименование статуса, Цвет (цветной квадрат), Аналитическая категория, Является рабочим статусом (Да/Нет), Организационная категория.
6. Повторите шаги 2-4 для создания всех необходимых статусов:
   Создайте статусы для всех состояний техники, которые нужно учитывать в работе.

ВАЖНО:
- Системные статусы нельзя удалить
- Для системных статусов нельзя изменить признак рабочего статуса
- Признак рабочего статуса влияет на расчет коэффициента использования

### Настройка горизонтов

Горизонты — это уровни рудника, которые используются для построения трехмерной карты с разными высотными отметками, привязки мест погрузки и разгрузки к конкретным уровням, отображения техники на правильном горизонте в режиме карты, построения дорожного графа для расчета маршрутов между точками и визуализации перемещения техники между уровнями через шахтные стволы.

1. В разделе «Справочники» выберите пункт «Горизонты»
   Откроется страница со списком горизонтов. Если горизонты еще не созданы, таблица будет пустой.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания нового горизонта.
3. Заполните поля формы:
   - В поле «Горизонт» введите название горизонта. Название отображается в карточках техники и мест на карте диспетчера, используется в выпадающих списках при создании мест работ
   - В поле «Высота» укажите высотную отметку в метрах. Высота определяет вертикальное положение горизонта в трехмерной модели, отображается в выпадающих списках, используется при загрузке подложки карты для привязки фона к горизонту
   - В поле «Цвет» выберите цвет для визуализации. Нажмите на поле выбора цвета, выберите цвет из палитры или введите код цвета в формате HEX. Цвет используется для отображения линий дорожного графа на карте диспетчера, цвет точек при добавлении новых узлов на граф и при привязке мест к графу
   - В поле «Шахта» выберите привязанные шахтные стволы. Нажмите на поле множественного выбора с поиском, выберите одну или несколько шахт из списка. Шахты определяют пути подъема и спуска между горизонтами. Можно создать новую шахту прямо в форме, начав вводить название, или переименовать шахту при редактировании горизонта
4. Нажмите кнопку «Сохранить»
   Горизонт будет добавлен в таблицу, появится сообщение об успешном создании.
5. Проверьте созданный горизонт в таблице:
   В таблице отображаются колонки: Горизонт, Высота, Шахта, Цвет. Цвет отображается как цветной квадрат в соответствующей колонке, высота отображается с единицей измерения «м».
6. Повторите шаги 2-4 для создания всех горизонтов рудника:
   Создайте горизонты для всех уровней, используемых на предприятии. Убедитесь что высоты указаны правильно для построения трехмерной модели.

ВАЖНО:
- Горизонты — это общая база данных для всего предприятия
- Все пользователи с правами доступа видят одни и те же горизонты
- Без горизонтов невозможно создать места погрузки и разгрузки
- Цвет горизонта влияет на визуализацию дорожного графа на карте
- Высота горизонта критична для правильного отображения трехмерной модели рудника

### Создание схемы дорог рудника

Редактор карты позволяет создавать схему дорожной сети рудника с узлами, дорогами и местами. Схему можно импортировать из готового JSON файла или нарисовать вручную с помощью инструментов.

1. Откройте раздел «Оперативная работа» → «Карта»
   Откроется страница «3D визуализация Шахты».
2. Нажмите кнопку «← Редактор» в правом верхнем углу
   Откроется редактор дорог с рабочими панелями: слева панель «Горизонты» и «Инструменты», в центре холст для рисования схемы, справа панель техники, которые раскроются при нажатии соответствующих иконок.
3. Создайте горизонт для размещения схемы дорог:
   - Слева активируйте панель «Горизонты»
   - Нажмите кнопку «+ Новый горизонт»
   - Заполните форму создания горизонта, где в поле «Название» необходимо ввести название уровня, в поле «Высота в метрах» указать высотную отметку, а в поле «Цвет» выбрать цвет для отображения горизонта на карте
   - Нажмите кнопку «Создать»
   Горизонт появится в списке и будет выбран автоматически.
4. Импортируйте готовую схему дорог (если есть JSON файл):
   - На панели «Горизонты» нажмите кнопку «Импорт»
   - Выберите способ импорта: вкладка «Из URL» для вставки ссылки на JSON файл со схемой или вкладка «JSON данные» для вставки содержимого JSON файла
   - Выберите куда импортировать схему: «Создать новый горизонт» для загрузки схемы на новый уровень или «В существующий горизонт» для выбора горизонта из списка
   - Отметьте галочку «Создавать вершины с метками» и укажите радиус метки в метрах
   - Нажмите кнопку «Импортировать»
   Дождитесь завершения импорта. Схема загрузится со всеми узлами, дорогами и местами.
5. Создайте схему дорог вручную (если готового файла нет):
   - Активируйте панель «Инструменты» слева
   - Создайте узлы дороги: нажмите кнопку «Узел» и нажмите на холст там где должна начинаться дорога, затем нажмите еще в нескольких местах вдоль дороги для создания узлов
   - Соедините узлы дорогой: нажмите кнопку «Дорога», затем нажмите на первый узел и на второй для создания линии, соедините все узлы по порядку
   - Добавьте места погрузки и разгрузки: нажмите кнопку «Место» и нажмите на холст там где находится место, затем нажмите кнопку «Правка» и нажмите на оранжевую точку места для открытия формы, где необходимо ввести название, выбрать тип и нажать «Сохранить»
   - Используйте дополнительные инструменты: «Переместить» для перетаскивания узла или места, «Удалить» для удаления объекта, «Лестница» для соединения узлов на разных горизонтах
6. Настройте GPS координаты для привязки схемы к реальным координатам:
   - Вверху нажмите кнопку «Настройки»
   - Отметьте галочку «Включить GPS режим (метрическая проекция)»
   - Найдите одну известную точку на карте и запишите её координаты на карте в метрах, затем узнайте GPS координаты этой точки из навигатора
   - Введите значения в форму калибровки и нажмите кнопку «Сохранить»
   Система будет преобразовывать GPS координаты техники в координаты карты.
7. Проверьте содержимое созданных горизонтов:
   - Вверху нажмите кнопку «Графы»
   - Откроется окно «Горизонты и содержимое» со списком всех горизонтов, где для каждого горизонта отображается название, высота в метрах и количество узлов, мест и лестниц
   - Нажмите на горизонт чтобы раскрыть содержимое с узлами, местами и тегами
   - Нажмите на координаты объекта чтобы показать его на карте
8. Повторите шаги 3-5 для создания схем на других горизонтах:
   Создайте схемы дорог для всех уровней рудника.

### Настройка участков

Участки — это производственные подразделения предприятия, которые отвечают за работу на определенных горизонтах. Они используются для группировки мест работ по подразделениям, распределения ответственности за выполнение работ, формирования отчетности по участкам, контроля работы подрядных организаций.

1. В разделе «Справочники» выберите пункт «Участки»
   Откроется страница со списком участков.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания нового участка.
3. Заполните поля формы:
   - В поле «Наименование подразделения» введите название производственного подразделения. Название отображается в таблице участков, используется в отчетах и для группировки данных
   - Установите галочку «Является подрядной организацией» если участок принадлежит подрядной организации. Используется для разделения собственных и подрядных участков в отчетности
   - В поле «Включает в себя» выберите один или несколько горизонтов из списка. Нажмите на поле множественного выбора с поиском, выберите горизонты на которых работает участок. Определяет зоны ответственности участка, используется для фильтрации мест работ по участкам
4. Нажмите кнопку «Сохранить»
   Участок будет добавлен в таблицу, появится сообщение об успешном создании.
5. Проверьте созданный участок в таблице:
   В таблице отображаются колонки: Наименование подразделения, Является подрядной организацией, Включает в себя.
6. Повторите шаги 2-4 для создания всех участков предприятия:
   Создайте участки для всех производственных подразделений и подрядных организаций.

ВАЖНО:
- Участки — это общая база данных для всего предприятия
- Участки используются для группировки мест работ и формирования отчетности
- Один участок может работать на нескольких горизонтах одновременно

### Настройка мест работ

Места работ — это точки на карте рудника, где происходят операции с грузом. Они используются для создания маршрутов техники, отображения на карте диспетчера, расчета расстояний и времени перемещения, учета остатков груза на местах погрузки и разгрузки, привязки меток позиционирования к конкретным точкам.

1. В разделе «Справочники» выберите пункт «Места»
   Откроется страница со списком мест работ.
2. Выберите тип мест из выпадающего списка в верхней части страницы:
   Нажмите на выпадающий список и выберите нужную группу: Места погрузки, Места разгрузки/перегрузки, Места стоянки/транзитные.
3. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания нового места. Набор полей зависит от выбранного типа места.
4. Заполните поля формы:
   - В поле «Наименование ПП» для мест погрузки, «Наименование ПР» для мест разгрузки/перегрузки или «Наименование» для остальных типов введите название места. Название отображается в таблице мест, в выпадающих списках при создании наряд-заданий, на карте диспетчера
   - Поле «Тип места» автоматически установлено в соответствии с выбранной группой. Для мест разгрузки/перегрузки можно выбрать между «Место разгрузки» и «Место перегрузки»
   - В поле «Местоположение» укажите координаты места на карте в формате широта, долгота. Используется для отображения места на карте диспетчера
   - В поле «Дата начала эксплуатации» укажите дату начала использования места в формате ГГГГ-ММ-ДД. Валидация: не может быть позже даты окончания
   - В поле «Дата конца эксплуатации» укажите дату окончания использования места. Валидация: не может быть раньше даты начала
   - В поле «Остаток, м³» укажите текущее количество груза на месте в кубических метрах. Для мест погрузки показывает сколько груза доступно для загрузки. Для мест разгрузки показывает сколько груза уже выгружено
   - В поле «Горизонт» выберите горизонт из списка созданных ранее. Определяет на каком уровне рудника находится место. Используется для отображения на трехмерной карте
   - В поле «Вид груза» для мест погрузки выберите тип груза из списка. Определяет какой груз загружается на этом месте. Используется для расчета веса и контроля соответствия груза между погрузкой и разгрузкой
   - В поле «Вместимость, м³» для мест разгрузки и перегрузки укажите максимальный запас груза в кубических метрах. Отображается в таблице мест
5. Нажмите кнопку «Сохранить»
   Место будет добавлено в таблицу, появится сообщение об успешном создании.
6. Проверьте созданное место в таблице:
   В таблице отображаются колонки в зависимости от типа места:
   - Для погрузки: Наименование ПП, Тип места, Местоположение, Дата начала эксплуатации, Дата конца эксплуатации, Остаток (м³), Горизонт, Вид груза, Дата изменения
   - Для разгрузки/перегрузки: Наименование ПР, Тип места, Местоположение, Дата начала эксплуатации, Дата конца эксплуатации, Вместимость (м³), Остаток (м³), Горизонт, Вид груза, Дата изменения
   - Для стоянки/транзита: Наименование, Тип места, Местоположение, Горизонт
7. Повторите шаги 2-5 для создания всех мест работ:
   Создайте места для всех типов операций, используемых на предприятии. Убедитесь что все места привязаны к правильным горизонтам.

ВАЖНО:
- Места работ — это общая база данных для всего предприятия
- Без мест погрузки и разгрузки невозможно создать наряд-задания
- Вид груза для места погрузки должен соответствовать виду груза места разгрузки в маршруте
- Остаток автоматически изменяется при выполнении рейсов техники

### Настройка меток позиционирования

Метки позиционирования — это физические устройства (маяки), которые устанавливаются в местах работ для определения местоположения техники. Они используются для отслеживания положения техники в руднике, автоматического определения прибытия техники на место погрузки или разгрузки, контроля перемещения техники между точками, мониторинга уровня заряда батарей меток.

1. В разделе «Справочники» выберите пункт «Метки»
   Откроется страница со списком меток позиционирования.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания новой метки.
3. Заполните поля формы:
   - В поле «ID-метки» введите уникальный идентификатор устройства. Используется для идентификации метки в системе, отображается в таблице меток
   - В поле «MAC-адрес» введите физический адрес устройства в формате XX:XX:XX:XX:XX:XX. Используется для связи с устройством, должен быть уникальным. Система автоматически проверяет формат ввода
   - В поле «Место установки» выберите место работ из списка. Нажмите на поле с поиском, выберите место где установлена метка. Определяет зону действия метки. При выборе места автоматически заполняются координаты и тип метки
4. Нажмите кнопку «Сохранить»
   Метка будет добавлена в таблицу, появится сообщение об успешном создании.
5. Проверьте созданную метку в таблице:
   В таблице отображаются колонки: ID-метки, MAC-адрес, Место установки, Координаты, Тип метки, Уровень заряда (%), Дата изменения. Поля Координаты, Тип метки, Уровень заряда и Дата изменения доступны только для чтения и заполняются автоматически.
6. Повторите шаги 2-4 для создания всех меток:
   Создайте метки для всех физических устройств, установленных в руднике.

ВАЖНО:
- MAC-адрес должен быть уникальным для каждой метки
- Метка должна быть привязана к месту работ для корректной работы позиционирования
- Уровень заряда обновляется автоматически при получении данных от устройства

### Настройка оборудования

Оборудование — это техника предприятия (самосвалы и погрузчики), которая выполняет перевозку груза. Используется для создания наряд-заданий и маршрутов, отслеживания перемещения техники на карте, учета выполненных рейсов и перевезенного груза, расчета производительности техники.

1. В разделе «Справочники» выберите пункт «Оборудование»
   Откроется страница со списком техники.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания новой единицы техники.
3. Заполните поля формы:
   - В поле «Гаражный номер» введите регистрационный номер техники на предприятии. Используется в названии техники, отображается в таблице оборудования
   - В поле «Тип техники» выберите тип оборудования: ШАС для самосвалов или ПДМ для погрузчиков. Определяет иконку техники на карте, доступные типы задач при создании наряд-заданий, группировку техники по типам
   - В поле «Модель» выберите модель из списка или создайте новую. Начните вводить название модели, если модели нет в списке появится опция создания новой. При выборе модели автоматически заполняются технические характеристики
   - В поле «Идентификатор/номер» введите серийный заводской номер техники. Отображается в таблице оборудования для идентификации конкретной единицы
   - В поле «Объем кузова/ковша, м³» укажите вместимость в кубических метрах. Автоматически заполняется из модели, можно изменить вручную. Используется для расчета количества рейсов
   - В поле «Грузоподъемность, т» укажите максимальную массу груза в тоннах. Автоматически заполняется из модели, можно изменить. Используется для контроля перегрузки
   - В поле «Максимальная скорость, км/ч» укажите максимальную скорость в километрах в час. Автоматически заполняется из модели, используется для расчета времени в пути
   - В поле «Объем бака, л» укажите емкость топливного бака в литрах. Автоматически заполняется из модели, используется для контроля расхода топлива
   - В поле «Дата ввода в эксплуатацию» укажите дату начала использования техники. Отображается в таблице оборудования
   - В поле «Дата вывода из эксплуатации» укажите дату окончания использования техники. Валидация: не может быть раньше даты ввода
4. Нажмите кнопку «Сохранить»
   Техника будет добавлена в таблицу, появится сообщение «Объект добавлен в таблицу». Техника сразу становится доступной для назначения заданий.
5. Проверьте созданную технику в таблице:
   В таблице отображаются колонки: Гаражный номер, Тип, Модель, Идентификатор/номер, Объем кузова/ковша (м³), Грузоподъемность (т.), Макс. скорость (км/ч), Объём бака (л.), Дата ввода в эксплуатацию, Дата вывода из эксплуатации.
6. Повторите шаги 2-4 для создания всей техники предприятия:
   Создайте все единицы оборудования, которые будут использоваться для перевозки груза.

ВАЖНО:
- Без техники невозможно создать наряд-задания

### Настройка персонала

Персонал — это сотрудники предприятия, которые работают с системой. Они используются для создания учетных записей пользователей, назначения ролей и прав доступа, контроля действий пользователей в системе, формирования отчетности по сотрудникам.

1. В разделе «Справочники» выберите пункт «Персонал»
   Откроется страница со списком сотрудников.
2. Нажмите кнопку «+ Добавить» в правом верхнем углу
   Откроется форма создания нового сотрудника.
3. Заполните поля формы:
   - В поле «Имя» введите имя сотрудника. Отображается в таблице персонала и в системе при входе пользователя
   - В поле «Фамилия» введите фамилию сотрудника. Используется вместе с именем для идентификации сотрудника
   - В поле «Отчество» введите отчество сотрудника. Дополняет полное имя сотрудника
   - В поле «Дата рождения» укажите дату рождения сотрудника. Используется для учета персональных данных
   - В поле «Телефон» введите контактный телефон сотрудника. Используется для связи с сотрудником
   - В поле «Почта» введите адрес электронной почты сотрудника. Используется для уведомлений и связи
   - В поле «Логин» введите имя пользователя для входа в систему. Должен быть уникальным, используется при авторизации
   - В поле «Пароль» введите пароль для входа в систему. Пароль чувствителен к регистру, храните его в безопасности
   - В поле «Должность» введите должность сотрудника. Отображается в карточке сотрудника, используется в отчетах
   - В поле «Подразделение» введите название подразделения где работает сотрудник. Используется для группировки сотрудников по отделам
   - В поле «Табельный номер» введите уникальный табельный номер сотрудника. Используется для идентификации сотрудника на предприятии, должен быть уникальным
   - В поле «Роль» выберите роль из списка. Определяет права доступа сотрудника к разделам и функциям системы
4. Нажмите кнопку «Сохранить»
   Сотрудник будет добавлен в таблицу, появится сообщение об успешном создании. Учетная запись пользователя создается автоматически.
5. Проверьте созданного сотрудника в таблице:
   В таблице отображаются колонки: Имя, Фамилия, Отчество, Дата рождения, Телефон, Почта, Логин, Пароль, Должность, Подразделение, Табельный номер, Роль.
6. Повторите шаги 2-4 для создания всех сотрудников:
   Создайте учетные записи для всех сотрудников, которые будут работать с системой.

#### Создание роли

1. Откройте раздел «Справочники» → «Роли»
2. Нажмите кнопку «+ Добавить»
   Справа откроется панель «Новая роль».
3. Введите название роли:
   В поле «Наименование роли» введите имя (например, «Диспетчер», «Оператор», «Администратор»).
4. Настройте права доступа для каждой формы системы:
   Ниже отображается таблица «Доступные формы» с двумя группами:
   Группа «Оперативная работа»:
   - Управление техникой
   - Карта
   - Карта (2D)
   - КРВ (Карта рабочего времени)
   - Управление рейсами
   Группа «Справочники»:
   - Роли
5. Для каждой формы выберите уровень доступа (радиокнопки):
   - Нет доступа — форма недоступна пользователю (по умолчанию)
   - Просмотр — только чтение данных без возможности изменения
   - Редактирование — полный доступ (создание, изменение, удаление)
6. Нажмите кнопку «Сохранить»
   Роль будет создана и появится в таблице ролей.

Примеры настройки ролей:
- Администратор — все формы на «Редактирование»
- Диспетчер — Оперативная работа на «Редактирование», Справочники на «Просмотр»
- Оператор — только Наряд-задание и Карта на «Просмотр»
- Наблюдатель — все формы на «Просмотр»

#### Изменение прав доступа роли

1. Откройте раздел «Справочники» → «Роли»
2. Найдите нужную роль в таблице
3. Откройте роль для редактирования:
   - Дважды кликните по строке с ролью
4. Справа откроется панель редактирования с названием роли:
   В панели отображается:
   - Поле «Наименование роли» с текущим названием (можно изменить)
   - Таблица «Доступные формы» с текущими правами доступа
5. Измените права доступа для нужных форм:
   Для каждой формы кликните на одну из трех радиокнопок:
   - Нет доступа — форма полностью недоступна
   - Просмотр — только чтение данных, без возможности изменения
   - Редактирование — полный доступ (создание, изменение, удаление)
6. Нажмите кнопку «Сохранить»
   Изменения сохраняются в базе данных. Пользователи с этой ролью увидят обновленные права доступа при следующем входе в систему или обновлении страницы.
