⚙️ ТЕХНОЛОГИИ:

  Frontend:
    - React 18 + TypeScript
    - Vite (сборщик)
    - Tailwind CSS (стили)
    - React Query (управление данными)
    - React Hook Form (формы)
    - Axios (HTTP клиент)

  Docker:
    - Development: Vite dev server + hot reload
    - Production: Nginx + оптимизированная сборка

  Инструменты:
    - ESLint (линтер)
    - date-fns (даты)
    - Lucide React (иконки)

🛑 ОСТАНОВКА:

  Windows:  stop.bat
  
  Linux/Mac:  ./stop.sh
  
  Командная строка:  docker-compose down
  
  Ctrl+C (если запущено в текущем окне)


🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

  1. Откройте DOCKER.md → "Решение проблем"
  2. Проверьте логи: docker-compose logs -f
  3. Проверьте backend: http://0.0.0.0:8001/docs
  4. Проверьте Docker: docker ps


💡 ПОЛЕЗНЫЕ КОМАНДЫ:

  # Статус контейнеров
  docker ps

  # Логи (real-time)
  docker-compose -f docker-compose.dev.yml logs -f

  # Пересборка
  docker-compose -f docker-compose.dev.yml up --build

  # Зайти в контейнер
  docker exec -it enterprise-frontend-dev sh

  # Остановка
  docker-compose down
